# ubiops-cli

Command Line Interface to interact with the UbiOps API (v2.1).

Includes:

- Managing projects
- Managing deployments
- Managing versions
- Managing pipelines
- Managing schedules
- Managing environment variables
- ZIP folder and deploy
- Making requests
- View logs
- View audit events
